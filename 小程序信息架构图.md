# 创意喵小程序信息架构图

> **版本**: V1.0
> **更新时间**: 2026-04-10
> **产品定位**: 连接商家与创作者的视频任务撮合平台（移动端）

---

## 一、产品结构总览

```
创意喵小程序
│
├── [Tab 1] 首页          /mobile/
│   └── 任务详情              /mobile/task/:id
│
├── [Tab 2] 灵感          /mobile/works
│   └── 作品详情              /mobile/work/:id
│
└── [Tab 3] 我的              /mobile/mine
    ├── 钱包                  /mobile/wallet
    │   └── 收益明细          /mobile/transactions
    ├── 我领取的任务          /mobile/my-claims
    ├── 我发布的任务          /mobile/my-tasks      (商家)
    ├── 设置                  /mobile/settings
    └── 登录/注册             /mobile/login
```

## 二、页面清单与路由

### Tab 1 - 任务大厅（首页）

| 页面 | 路由 | 功能说明 |
|------|------|----------|
| 首页 | `/mobile/` | 推荐任务流、搜索、分类筛选、无限滚动 |

- 顶部搜索框（关键词搜索，500ms 防抖）

- 排序选项（最新/高价优先/低价优先）
- 任务卡片流（双列瀑布流）
- 底部 Tab Bar

## 六、核心交互规范

### 6.1 导航结构

- **底部 Tab Bar**: 3 个主 Tab 固定底部全局导航
- **顶部导航栏**: 二级页使用返回按钮 + 标题 + 占位符
- **跳转方式**: 点击卡片跳转详情页，不在新窗口打开

### 6.2 列表加载

- **首屏**: 服务端渲染首屏数据
- **翻页**: 无限滚动，触底加载下一页
- **刷新**: 下拉刷新当前列表
- **防抖**: 搜索输入 500ms 防抖

### 6.3 状态处理

| 状态 | 处理方式 |
|------|----------|
| 加载中 | 显示 loading 指示器 |
| 空列表 | 显示空状态插画和提示文案 |
| 加载失败 | 显示错误状态和重试按钮 |
| 已登录 | 显示用户信息和操作入口 |
| 未登录 | 引导用户去登录页 |

### 6.4 移动端适配

- 视口: `width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no`
- 禁止双击缩放
- 禁止长按弹出菜单
- 图片懒加载
- 触摸事件优化

## 八、技术架构

### 8.1 目录结构

```
web/
├── templates/mobile/           # 移动端模板
│   ├── layout.html             # 基础布局
│   ├── index.html              # 任务大厅
│   ├── works.html              # 过审作品
│   ├── mine.html               # 我的
│   ├── task_detail.html        # 任务详情
│   ├── work_detail.html        # 作品详情
│   ├── wallet.html             # 钱包
│   ├── transactions.html       # 收益明细
│   ├── my_claims.html          # 我领取的任务
│   ├── settings.html           # 设置
│   ├── login.html              # 登录
│   └── components/             # 组件片段
│       └── task_card.html
│
└── static/mobile/             # 移动端静态资源
    ├── css/
    │   └── mobile.css          # 移动端样式
    └── js/
        └── mobile.js           # 移动端脚本
```

### 8.2 前端技术栈

- **模板**: Gin 原生模板
- **样式**: 原生 CSS + CSS 变量（小红书风格）
- **脚本**: 原生 JavaScript（ES6+）
- **图标**: Emoji 字符
- **图片**: 懒加载 + 占位图

### 8.3 认证方式

- 登录状态存储于 `localStorage.token`
- 同时同步到 Cookie（用于服务端渲染页面认证）
- 敏感操作通过 `Authorization: Bearer <token>` Header 传递

## 九、视觉风格

### 9.1 设计语言

- **风格定位**: 小红书风格、偏内容化、偏 app 风格
- **色调**: 红色系主色调（#FF2442）
- **圆角**: 大圆角卡片
- **阴影**: 轻阴影
- **留白**: 充足留白，信息分组清晰

### 9.2 色彩系统

```css
--mobile-primary: #FF2442;       /* 主色 */
--mobile-bg: #F5F5F5;           /* 背景色 */
--mobile-card-bg: #FFFFFF;      /* 卡片背景 */
--mobile-text: #333333;         /* 主文字 */
--mobile-text-secondary: #999; /* 次要文字 */
--mobile-border: #EEEEEE;      /* 边框色 */
```

### 9.3 间距规范

- 页面边距: 12px ~ 16px
- 卡片间距: 12px
- 卡片内边距: 16px
- 元素间距: 8px ~ 12px
